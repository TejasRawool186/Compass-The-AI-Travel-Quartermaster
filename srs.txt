Compass — Final Hackathon SRS
“The AI Travel Quartermaster”
1. Project Overview

Compass is a pirate-themed AI travel planning agent built for the Coral Personal Agent Track. It helps users discover and plan trips using natural language by intelligently combining flights, hotels, weather forecasts, calendar schedules, and travel budgets through Coral’s cross-source SQL engine.

Instead of switching between multiple travel websites, weather apps, and calendars, users can simply ask:

“Find me a sunny beach getaway next weekend under $400.”

Compass uses Coral to join live travel-related data sources in a single SQL query and generates personalized travel recommendations with AI-powered explanations.

The application uses:

Flask backend
React + Tailwind frontend
Coral query engine
MCP integration
Custom Coral source specs
AI natural language interface

The project runs 100% locally and demonstrates Coral’s most powerful capabilities:

Cross-source SQL joins
Schema learning
MCP tools
Query caching
Custom source connectors
Local-first execution
2. Problem Statement

Travel planning is fragmented and time-consuming.

Users typically:

search flights separately,

Compass solves this by combining all travel intelligence into one AI-driven experience powered by Coral SQL.

4.2 Cross-Source Coral SQL Intelligence

Compass joins:

Flights
Hotels
Weather
Calendar
Budget

using one SQL query.

Example Hero Query
SELECT 
    f.destination_city,
    h.hotel_name,
    w.forecast,
    b.remaining_budget
FROM flights.search f
JOIN hotels.search h
ON h.city = f.destination_city
JOIN weather.forecast w
ON w.city = f.destination_city
LEFT JOIN budget.allowance b
ON b.category = 'travel'
WHERE w.forecast = 'Clear';
